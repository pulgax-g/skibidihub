🧭 MENÚ — DESCRIPCIÓN A ALTO NIVEL 
──────────────────────────────────────────────────────────────────────────────
{1} RANSOMWARE
⇒ Sugiere software que cifra y extorsiona datos. Riesgo: pérdida y exfiltración de información.

{2} KILL PC
⇒ Sugiere una acción destinada a apagar o inutilizar un equipo. Riesgo: interrupción/daño.

{3} RAT
⇒ Acrónimo de Remote Access Trojan. Sugiere control remoto no autorizado y acceso persistente.

{4} FAKE BLUESCREEN
⇒ Simulación de una pantalla azul de error (BSOD). Usado para alarmar usuarios o ocultar actividad.

{5} MID KILL PC
⇒ Variante que también sugiere daño o interrupción del sistema.

{6} remote controll pc
⇒ Control remoto de equipos; potencial para acceso no autorizado.

{7} bruteforce pc pass
⇒ Intento automatizado de adivinar contraseñas locales. Actividad intrusiva/ilegal.

{8} bruteforce wifi pass
⇒ Intento de adivinar contraseñas de redes Wi-Fi. Actividad intrusiva/ilegal.

{11} discord
⇒ Referencia a Discord (comunicación/difusión). Puede ser benigno o usarse para coordinación.

{22} exit
⇒ Salir del menú. Acción de interfaz, inofensiva.
